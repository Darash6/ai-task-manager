# 🧠 AI Task Manager

Sistema de gerenciamento de tarefas com foco em organização e produtividade, utilizando conceitos de backend e integração com IA.

---

## 🚀 Tecnologias utilizadas

* Python
* FastAPI / Flask 
* Docker 
* Git e GitHub

---

## 📌 Funcionalidades

* Criar tarefas
* Listar tarefas
* Atualizar status
* Organização simples e eficiente

---

## ⚙️ Como rodar o projeto

```bash
# Clonar repositório
git clone https://github.com/darash6/ai-task-manager.git

# Entrar na pasta
cd ai-task-manager

# Instalar dependências
pip install -r requirements.txt

# Rodar no terminal
docker-compose up --build

```
